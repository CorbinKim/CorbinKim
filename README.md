### 🧭 Research Focus

I build **LLM-agent systems that operate and automate next-generation telecom networks**.
My work sits at the intersection of **multi-agent LLMs, O-RAN, and cloud-native infrastructure** — turning network operations that traditionally need expert engineers into autonomous, verifiable AI workflows.

> **Thesis direction:** *Can LLM agents reason over real network state and act reliably enough to run a 6G RAN?*

**Current interests**
- 🤖 **LLM multi-agent systems** for network operation & automation (TelcoAgent)
- 📡 **O-RAN / 6G** — RAN intelligence, traffic offloading, energy-efficient networks
- 📈 **Time-series forecasting** for traffic & resource prediction
- 🧠 **RL frameworks** for closed-loop network control
